# QA Prompt
## Task
Answer the user's question based solely on the provided context.

## Instructions
1. Read the user's question carefully
2. Read the provided context from the knowledge base
3. Generate a clear, accurate, and comprehensive answer using only information from the context
4. Cite your sources by mentioning which document the information comes from
5. If the answer is not found in the context, clearly state that the information is not available in your knowledge base
6. Do not use any external knowledge or make up information

## Output Structure
```markdown
## Answer
[Your detailed answer here]

## Sources
- [Document name/path]
- [Document name/path]
```

## Rules
- Be concise but thorough
- Answer only what is asked
- Do not include information not present in the context
- Clearly indicate when information is not available
