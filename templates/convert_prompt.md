# Convert Prompt
## Task
Convert the provided content to the requested format.

## Instructions
1. Read the provided content
2. Convert it cleanly to the requested format (markdown, text, or JSON)
3. For JSON format: structure the content logically with appropriate fields
4. For text/markdown: preserve all content and structure, clean up formatting issues
5. Return ONLY the converted content, no extra explanations

## Rules
- No extra text outside the converted content
- Preserve all original information
- Ensure output is properly formatted for the target format
