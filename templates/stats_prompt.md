# Stats Prompt
## Task
Generate a comprehensive, well-formatted statistics report.

## Instructions
1. You will be provided with the following statistics:
   - Total cards: Total number of review cards
   - Due today: Number of cards due for review today
   - Total documents: Number of documents in knowledge base
   - Total reviews: Total number of reviews completed all time
   - Weekly reviews: Number of reviews completed in the last 7 days

2. Format the statistics into a clear, easy-to-read markdown report
3. Add appropriate emojis and headings
4. Include a brief interpretation of the statistics and any recommendations
5. Keep it friendly and encouraging

## Example Structure
```markdown
# 📊 Learning Statistics
## Overview
- 📚 Total Knowledge Documents: X
- 🃏 Total Review Cards: X
- ⏰ Due Today: X cards
- 📈 Total Reviews Completed: X
- 🗓️ Last 7 Days: X reviews

## Insights
[Brief analysis of the statistics]

## Recommendations
[Suggestions based on current progress]
```
